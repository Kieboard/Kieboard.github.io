# kieboard.me — Cyber Security Portfolio

First ever Personal portfolio site for Kieran (Kieboard) — IT professional transitioning into cyber security, documenting the journey through projects, certs, and CTF writeups.

Live at **[kieboard.me](https://kieboard.me)**

---

## What's on the site

- **About** — background, skills, and current focus
- **Certs** — ITIL 4 Foundation, Git Foundations, CompTIA Security+ (in progress)
- **Experience** — IT career timeline with a cyber security lens
- **Projects** — TryHackMe rooms, tools & automation, Open University modules
- **Writeups** — Live CTF writeups pulled nightly from GitBook via GitHub Actions
- **KieOS Terminal** — interactive terminal with games, AI chat, and easter eggs

---

## Tech

Built with vanilla HTML, CSS, and JavaScript — no frameworks.

- GitHub Pages hosting
- Cloudflare DNS + email obfuscation
- Swiper.js for the cert slider
- marked.js for markdown rendering
- GitHub Actions for automated GitBook writeup fetching
- FormSubmit for the contact form

---

## CTF Writeups
